## Brief overview
Este arquivo contém diretrizes sobre o uso de comandos CLI para agilizar o desenvolvimento, especificamente quais comandos podem ser executados com auto-aprovação.

## Comandos CLI e Auto-aprovação
- Comandos que começam com `npm run mdf` (make domain file) são permitidos para auto-aprovação.
  - Exemplo de trigger: `npm run mdf Admin Api admin`
- Comandos que começam com `npm run md` (make domain) são permitidos para auto-aprovação.
  - Exemplo de trigger: `npm run md Users --with Modals Panels`
