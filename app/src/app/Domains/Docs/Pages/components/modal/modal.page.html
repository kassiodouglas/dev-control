Visão Geral

O app-modal é um componente para exibir conteúdo em uma camada sobre a aplicação. Ele é controlado pelo ModalService, que gerencia sua visibilidade (abrir/fechar).

O componente utiliza projeção de conteúdo para o título, subtítulo, corpo e ações, oferecendo flexibilidade para diferentes casos de uso.

import { Component, inject } from '@angular/core';
import { ModalService } from '@shared/Components/modal/Services/modal.service';

@Component({
  selector: 'app-sua-pagina',
  // ...
})
export class SuaPaginaComponent {
  private modalService = inject(ModalService);

  openModal(id: string) {
    this.modalService.open(id);
  }

  closeModal(id: string) {
    this.modalService.close(id);
  }
}

API do Componente

Inputs

Nome Tipo Padrão Descrição
id string - (Obrigatório) Um identificador único para o modal.
size 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl' | 'full' 'md' Define a largura máxima do modal.

Slots de Conteúdo (Content Projection)

Seletor Descrição
[title] Projeta o conteúdo no título do modal.
[subtitle] Projeta o conteúdo no subtítulo do modal.
[actions] Projeta o conteúdo na área de ações (rodapé) do modal.
(default) Qualquer conteúdo sem um seletor específico é projetado no corpo do modal.
