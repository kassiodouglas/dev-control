Primary Button Component

O componente de botão primário é usado para ações principais em uma página.

Uso

Para usar o componente de botão primário, basta adicionar o seletor `comp-primary-button` ao seu template.

Exemplo:


Clique Aqui

Carregando...

Desabilitado

Inputs

  • isLoading (boolean): Se o botão está em estado de carregamento. O padrão é `false`.
  • disabled (boolean): Se o botão está desabilitado. O padrão é `false`.
  • type ('button' | 'submit' | 'reset'): O tipo do botão. O padrão é `'button'`.
  • loadingText (string): O texto a ser exibido quando o botão está em estado de carregamento. O padrão é `'Aguarde...'`.
