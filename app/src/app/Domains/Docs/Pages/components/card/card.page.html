Visão Geral

O comp-card é um componente versátil para exibir conteúdo em um contêiner estilizado. Ele suporta a projeção de conteúdo para o cabeçalho, corpo e rodapé, permitindo uma grande flexibilidade no layout.

API do Componente

Inputs

Nome Tipo Padrão Descrição
customClass string '' Classes CSS customizadas para aplicar ao contêiner principal do card.

Slots de Conteúdo (Content Projection)

Seletor Descrição
[header] Projeta o conteúdo no cabeçalho do card.
[body] Projeta o conteúdo no corpo do card.
[footer] Projeta o conteúdo no rodapé do card.
