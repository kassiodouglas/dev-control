Visão Geral

O componente comp-breadcrumb é usado para exibir a hierarquia de navegação, permitindo que os usuários entendam sua localização atual na aplicação.

Exemplo de Uso

Código

HTML

TypeScript


import { Component } from '@angular/core';
import { BreadcrumbItem } from '...'; // Importe a interface

@Component({
  selector: 'app-sua-pagina',
  templateUrl: './sua-pagina.component.html',
})
export class SuaPaginaComponent {
  exampleBreadcrumbs: BreadcrumbItem[] = [
    { label: 'Home', url: '/' },
    { label: 'Produtos', url: '/produtos' },
    { label: 'Categoria', url: '/produtos/categoria' },
    { label: 'Produto' },
  ];
}

API do Componente

Inputs

Nome Tipo Descrição
items BreadcrumbItem[] Uma lista de objetos BreadcrumbItem, onde cada objeto representa um nível da navegação. A interface BreadcrumbItem contém as propriedades label (string, obrigatório) e url (string, opcional).
