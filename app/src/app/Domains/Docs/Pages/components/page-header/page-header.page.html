Visão Geral

O componente comp-page-header é usado para criar um cabeçalho de página padronizado, incluindo título, subtítulo e breadcrumbs para navegação.

Exemplo de Uso

Código

HTML

TypeScript


import { Component } from '@angular/core';
import { BreadcrumbItem } from '...'; // Importe a interface

@Component({
  selector: 'app-sua-pagina',
  // ...
})
export class SuaPaginaComponent {
  exampleBreadcrumbs: BreadcrumbItem[] = [
    { label: 'Home', url: '/' },
    { label: 'Configurações' },
  ];
}

API do Componente

Inputs

Nome Tipo Padrão Descrição
title string '' O título principal do cabeçalho.
subtitle string '' O subtítulo opcional exibido abaixo do título.
breadcrumbs BreadcrumbItem[] [] Uma lista de objetos BreadcrumbItem para exibir a navegação.
