Visão Geral

O comp-input-field é um componente de formulário encapsulado que simplifica a criação de campos de entrada (inputs) com validação, rótulos, ícones e mensagens de erro. Ele deve ser usado dentro de um FormGroup do Angular Reactive Forms.

Exemplo de Uso

Código

HTML

TypeScript


import { Component, inject } from '@angular/core';
import { FormBuilder, Validators } from '@angular/forms';

@Component({
  selector: 'app-sua-pagina',
  // ...
})
export class SuaPaginaComponent {
  private fb = inject(FormBuilder);

  myForm = this.fb.group({
    name: ['', [Validators.required, Validators.minLength(3)]],
    email: ['', [Validators.required, Validators.email]],
    password: ['', [Validators.required, Validators.minLength(6)]],
  });
}

API do Componente

Inputs

Nome Tipo Padrão Descrição
label string '' O rótulo exibido acima do campo.
controlName string '' (Obrigatório) O nome do controle no FormGroup.
placeholder string '' O texto de placeholder do campo.
type string 'text' O tipo do input (ex: 'text', 'email', 'password').
icon string '' O nome do ícone do Material Icons a ser exibido dentro do campo.
disabled boolean false Desabilita o campo de entrada.
