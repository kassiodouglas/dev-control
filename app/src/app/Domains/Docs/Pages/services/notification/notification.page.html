Visão Geral

O NotificationService é um serviço injetável que permite exibir mensagens de notificação (toasts) para o usuário. É ideal para fornecer feedback sobre ações, como sucesso, erro, aviso ou informação.

Exemplo de Uso

Código

TypeScript


import { Component, inject } from '@angular/core';
import { NotificationService } from 'src/app/Shared/Services/notification.service';

@Component({
  selector: 'app-sua-pagina',
  templateUrl: './sua-pagina.component.html',
})
export class SuaPaginaComponent {
  private notificationService = inject(NotificationService);

  showSuccess() {
    this.notificationService.success('Esta é uma mensagem de sucesso!');
  }

  showFailure() {
    this.notificationService.failure('Esta é uma mensagem de falha.');
  }

  showWarning() {
    this.notificationService.warning('Esta é uma mensagem de aviso.');
  }

  showInfo() {
    this.notificationService.info('Esta é uma mensagem de informação.');
  }
}

API do Serviço

Métodos

Nome Parâmetros Descrição
success(message: string) message: A mensagem a ser exibida. Exibe uma notificação de sucesso.
failure(message: string) message: A mensagem a ser exibida. Exibe uma notificação de falha (erro).
warning(message: string) message: A mensagem a ser exibida. Exibe uma notificação de aviso.
info(message: string) message: A mensagem a ser exibida. Exibe uma notificação de informação.
